# 业余无线电考试学习网站

一个响应式的业余无线电考试学习网站，支持刷题、模拟考试和错题本功能。

## 功能特点

- 📱 **响应式布局**：手机、平板、电脑都能完美适配
- 📝 **刷题模式**：逐题练习，显示答案和解析
- 🎯 **模拟考试**：随机 40 题（32 单选 +8 多选），40 分钟计时，自动评分
- 📚 **错题本**：自动记录错题，可反复练习
- 💾 **本地存储**：使用 localStorage 保存错题和进度
- 🔍 **知识点分类**：11 个知识点类别，针对性练习

## 使用方法

1. 确保已安装 Node.js 和 npm
2. 在项目根目录执行以下命令：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

应用将在 `http://localhost:3001` 上运行（如果3000端口被占用会自动切换到其他端口）

## 技术栈

- **React 18** - 用户界面库
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **React Router** - 页面路由
- **localStorage** - 本地数据持久化

## 原始数据

### 文件

| 文件 | 说明 |
|------|------|
| docs/cq_class_A.txt | 中国业余无线电 A 类考试题库 |
| docs/cq_class_B.txt | 中国业余无线电 B 类考试题库 |
| docs/cq_class_C.txt | 中国业余无线电 C 类考试题库 |

### 格式解释

```
[J] 内部编号
[P] 所属章节篇
[I] 多选组编号，MC=多选，MC1=单选，MC2=双选，MC3=三选，MC4=全选
[Q] 问题题目内容
[T] 答案，多字符=多选
[A] 答案 A 内容
[B] 答案 B 内容
[C] 答案 C 内容
[D] 答案 D 内容
[F] 附件图片（部分题目可能没有）
```

## 题目数据格式

```javascript
{
    id: "[Lk0001:MC1-0001]",           // 内部编号:多选组编号（LX:MC2=0103）
    topic: "无线电管理",      // 知识点
    content: "题目内容...",   // 题干
    options: ["A. 选项 1", "B. 选项 2", ...], // 选项
    answer: "AC",           // 正确答案（多选）
    explanation: "解析..."    // 答案解析
}
```

## 开发工具

### Python 脚本

项目包含以下 Python 脚本用于题库管理：

#### 1. parse_data_2_json.py - 解析题库

```bash
# 解析 A 类题库
python3 parse_data_2_json.py -i docs/cq_class_A.txt -o questions_class_A.js

# 解析 B 类题库
python3 parse_data_2_json.py -i docs/cq_class_B.txt -o questions_class_B.js

# 解析 C 类题库
python3 parse_data_2_json.py -i docs/cq_class_C.txt -o questions_class_C.js

# 显示帮助
python3 parse_data_2_json.py --help
```

参数：
- `--input, -i`: 输入 txt 文件路径（默认：cq_class_A.txt）
- `--output, -o`: 输出 JS 文件路径（可选）

## 考试规则

### A 类考试

- 单选题 32 题，多选题 8 题
- 答题时间 40 分钟
- 答对 30 题为合格

> 参加 A 类操作技术能力验证的人员，应当熟悉无线电管理规定，具有一定的业余无线电台操作技术能力。

### B 类考试

- 单选题 45 题，多选题 15 题
- 答题时间 60 分钟
- 答对 45 题为合格

> 参加 B 类操作技术能力验证的人员，应当依法取得业余无线电台执照 6 个月以上，且具有相应的实际操作经验。

### C 类考试

- 单选题 70 题，多选题 20 题
- 答题时间 90 分钟
- 答对 70 题为合格

> 参加 C 类操作技术能力验证的人员，应当依法取得载明 30MHz 以下频段的业余无线电台执照 18 个月以上，且具有相应的实际操作经验。
