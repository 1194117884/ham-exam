# 业余无线电 A 类考试学习网站

一个响应式的业余无线电 A 类考试学习网站，支持刷题、模拟考试和错题本功能。

## 功能特点

- 📱 **响应式布局**：手机、平板、电脑都能完美适配
- 📝 **刷题模式**：逐题练习，显示答案和解析
- 🎯 **模拟考试**：随机 100 题，60 分钟计时，自动评分
- 📚 **错题本**：自动记录错题，可反复练习
- 💾 **本地存储**：使用 localStorage 保存错题和进度

## 使用方法

直接在浏览器中打开 `index.html` 文件即可使用。

```bash
# 方式 1：直接打开
open index.html

# 方式 2：使用本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 技术栈

- **HTML5** - 单文件结构
- **Tailwind CSS** (CDN) - 响应式样式
- **原生 JavaScript** - 无框架依赖
- **localStorage** - 本地数据持久化

## 题目数据格式

```javascript
{
    id: "[A001]",           // 题目编号
    topic: "无线电管理",     // 知识点
    content: "题目内容...",  // 题干
    options: ["A. 选项 1", "B. 选项 2", ...], // 选项
    answer: "A",            // 正确答案
    explanation: "解析..."   // 答案解析
}
```

## 扩展题库

编辑 `index.html` 中的 `questionsData` 数组，按照上述格式添加更多题目。
